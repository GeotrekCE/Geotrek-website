Technical info

Trail sections as a common reference

All objects are described based on the network of trail sections, using linear referencing.

This allows to preserve geometrical and topological consistency while linking all your data together using overlays on the network.

Full power of PostGIS 2

Leverage the 3rd dimension

Objects are edited in 2D and their geometry is automatically mapped to a 3D digital elevation model stored in the database in raster format.

Topological trail management

The application is based on a topological network of trail sections, which allows clipping edges at their intersections and snapping to network nodes.
